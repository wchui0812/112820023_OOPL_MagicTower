# Game Introduction

《魔塔》是一款經典的 2D 策略 RPG 遊戲，玩家扮演一名勇者，目標是爬上擁有不同配置的塔頂救出公主。
不同於傳統 RPG 的隨機戰鬥，魔塔的戰鬥結果是完全可預測的，因為玩家必須在有限的資源（血量、鑰匙、攻防數值）下，精密計算戰鬥損耗，決定升級順序，以最小傷害打通魔塔救出公主。

[遊戲連結](https://www.youtube.com/watch?v=RUJ3zd6NrSU)
